# AI Customer Support Email Responder
## Version 1.1

Status: ✅ Stable Release

---

# Overview

Version 1.1 introduces AI-powered issue classification and intelligent routing.

Instead of using a single AI response for every customer email, the workflow now classifies the customer's issue and routes it to specialized AI prompts.

---

# New Features

## AI Issue Classification

- Automatic customer issue detection
- AI-powered email analysis
- Intelligent category prediction

Supported Categories

- Refund
- Order Status
- Cancellation
- Damaged Item
- Wrong Item
- General Inquiry

---

## Smart Routing

Added Rule-based Router

Each issue is automatically routed to a dedicated response generator.

---

## Specialized AI Prompts

Separate AI prompts were created for:

- Refund requests
- Order status enquiries
- Cancellation requests
- Damaged products
- Wrong item deliveries
- General enquiries
