# Cancellation Policy

## Cancellation Eligibility

Customers may cancel an order only if it has not yet been shipped.

Once the order has been shipped, it can no longer be cancelled and must follow the return or refund process if applicable.

---

## Information Required

Support agents should request:

- Order Number
- Email used to place the order
- Whether the customer wants to cancel the entire order or only specific items

---

## Cancellation Review

Support agents must verify the current order status before confirming cancellation.

Cancellation should never be guaranteed until verification is complete.

---

## If the Order Has Already Shipped

If shipment has already begun:

- Explain that cancellation is no longer possible.
- Provide return instructions if the customer no longer wants the item.
- Inform the customer about the refund process after the return is received.

---

## Refund After Cancellation

If the cancellation is approved before shipment:

- A refund will be issued to the original payment method.
- Processing time depends on the customer's payment provider.

---

## Customer Communication

Support should:

- Be polite and empathetic.
- Clearly explain why cancellation may not be possible.
- Never promise cancellation before checking the order status.
- Offer the next available solution when cancellation cannot be completed.
